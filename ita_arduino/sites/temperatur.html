DHT22 - Die Beschreibung

Sensoren zur Messung der Luftfeuchtigkeit sind oft nicht gerade billig. Ein recht preiswerter und einfach verwendbarer Sensor ist der DHT22. Der DHT22 ist ein zuverlässiger Sensor zum ermitteln von Temperatur und Luftfeuchtigkeit. Da der Sensor sowohl mit 3,3V als auch 5V betrieben werden kann, eignet er sich zum Anschluss an alle gängige Boards wie Arduino, RN-Control, Raspberry Pi und alle anderen Microcontroller. Neben der Betriebsspannung muss lediglich ein Port mit dem Sensor verbunden werden. Die Ausgabe erfolgt seriell als digitale Bitfolge. Somit eignet sich der Sensor ideal um das Raumklima zu überwachen oder eine Wetterstation aufzubauen.

Technische Daten

  • Digital Temperatur und Luftfeuchtigkeit ermitteln
  • Chip Typ DHT22
  • Betriebsspannung: DC 3,3 - 5V
  • Luftfeuchtigkeitsmessbereich: 0 bis 100% relative Luftfeuchtigkeit
  • Feuchtemessgenauigkeit: ± 2% RH
  • Temperaturbereich: -40 bis +80 ˚C
  • Temperaturmessgenauigkeit: ± 0,5 ˚C
  • Single-Bus-Digigtalsignalausgang, bidirektionale serielle Daten
  • zahlreiche Beispielprogramme für gängige Boards im Internet verfügbar
  • Maße (HxBxT): 28mm x 12mm x 10mm

DS18B20

Der DS18B20 digitale Temperatursensor ist ein Sensor, der über den sogenannten 1-Wire Bus Daten überträgt. Wie der Name schon sagt, wird für die Kommunikation nur eine Leitung benötigt (und GND). Um die einzelnen Sensoren identifizieren zu können, ist in jedem Sensor eine eindeutige 64-Bit Adresse auf dem ROM gespeichert. Für die 1-Wire Kommunikation kann ein beliebiger digitaler Input Pin verwendet werden, auf dem mit einem 4.7kΩ pull-up Widerstand das Potenzial hochgezogen ist. (Der interne Pull-up Widerstand reicht nicht aus). Um den Sensor verwenden zu können, wird die OnWire und die Dallas Temperatur Library benötigt. Diese ermöglicht es uns einen Alarm für eine Temperatur Unter- oder Überschreitung zu setzen. Dieses Alarm Bit wird bei jeder Temperatur Messung neu bewertet.

Technische Daten

  • Betriebsspannung: 3.0 bis 5.5V
  • Genauigkeit Temperatur: ± 0.5˚C bis ± 0.0625 ˚C
  • Messbereich Temperatur: -55 bis +125 ˚C
  • Schnittstelle: 1-Wire Bus
  • Ausführungen: TO-92, SO 150 mil., μSOP
  • Adresse: 64-Bit Serien Nummer im ROM
  • 9-bit bis 12-bit Messauflösung frei wählbar

DS18B20 - Der Sketch

01 - #include <DallasTemperature.h>

02 - #include <OneWire.h>

03 - #define LKTemp2 4

04 - OneWire oneWire(LKTemp2);

05 - DallasTemperature sensors(&oneWire);

06 - void setup()

07 - {

08 - Serial.begin(9600);

09 - Serial.println("LK-Temp2 Temperaturmessung");

10 - sensors.begin(); // Sensor wird initialisiert

11 - }

12 - void loop()

13 - {

14 - sensors.requestTemperatures();

15 - Serial.print("Temperatur: ");

16 - Serial.print(sensors.getTempCByIndex(0));

17 - Serial.println(" C");

18 - delay(1000);

19 - }
