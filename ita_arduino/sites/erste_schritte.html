Dies ist der Mikrokontroler Arduino AT-Mega 2560

Um diesen Controller benutzen zu können benötigt man zum die Entwicklungsumgebung von Arduino, welche kostenfrei unter https://www.arduino.cc/en/software herunter geladen werden kann. Dort müssen noch kleine Einstellungen vorgenommen werden. Diese entnehmen Sie aus den Unterlagen im Dowwnload Bereich unter Erste Schritte mit dem Arduino.

Die Programme welche für den Arduino geschrieben werden, werden Sketche genannt. Diese Sketche bestehen im großen und ganzen aus drei Teilen: Variablen Deklaration und Initalisierung, Setup() und Loop()

Im Variablenbereich werden die Variablen erzeugt und Bibliotheken importiert, die in diesem Sketch benötigt werden. Hier können wir aus einer Gruppe von Variablentypen wählen, diese benötigen unterschiedlich viel Speicherplatz. Da der Arduino einen begrenzten Speicherplatz hat sollte immer der passende Variablentyp genutzt werden:

int
Integer, 16bit (=216) bzw. 2 Bytes, Wertebereich -32468 bis 32767, positive und negative ganze Zahlen
unsigned int
2 Bytes, Wertebereich 0 bis 65535, positive ganze Zahlen
long
4 Bytes, Wertebereich -2147483648 bis 2147483647, negative und positive ganze Zahlen
unsigned long
4 Bytes, Wertebereich 0 bis 4294967295, positive ganze Zahlen
float
32 bit (=232) bzw. 4 Bytes als Dezimalzahl im Bereich -3.4028235E+38 bis 3.4028235E+38. Wenn Sie mit floats rechnen,
müssen Sie ein Dezimalzeichen hinzufügen, andernfalls wird es als int behandelt. Floats haben eine Genauigkeit von nur 6-7 Dezimalstellen.
boolean
1 Byte, false (0) oder true (1), Wahr/Falsch-Werte
char
1 Byte, Wertebereich -128 bis 127, repräsentiert ein Zeichen gemäß ASCII-Tabelle
byte
1 Byte, Wertebereich 0 bis 255, ähnlich wie char

Syntax
Variablen werden klein geschrieben, dürfen nicht mit einer Zahl beginnen und sollten immer Aussagekräftig bezeichnet werden.
Sollte eine Variable aus mehreren Worten bestehen sind diese mit einem Unterstrich zu verbinden. Die Variablen sollten auch immer mit einem
Startwert versehen werden.

Beispiel: variable_mit_mehreren_worten = 0;

Im Setup() - Bereich wird den einzelnen Variablen eine Aufgabe zugeteilt und der serielle Monitor eingestellt

Im Loop() - Bereich wird das eigentliche Programm geschrieben.
